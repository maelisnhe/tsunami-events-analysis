# Tsunami Events Analysis

## Project overview
FR  
Ce projet de Data Analysis a pour objectif d'analyser des événements de tsunamis afin de mieux comprendre leur distribution temporelle, leur répartition géographique et certaines caractéristiques physiques ou humaines associées à leur impact.

EN  
This Data Analysis project aims to analyse tsunami events in order to better understand their temporal distribution, geographic spread, and several physical or human-related characteristics associated with their impact.

## Dataset
- Name: `tsunami_dataset.csv`
- Source: Provided project dataset
- The CSV file must be placed in the same folder as the notebook and the dashboard.

## Installation
Install the required Python libraries with:

```bash
pip install -r requirements.txt
```

## Run the notebook

```bash
jupyter notebook NoheEstradaMaelis_notebook.ipynb
```

## Run the dashboard

```bash
python NoheEstradaMaelis_dashboard.py
```

Then open the local URL displayed by Dash in your browser.

## Indicators
1. Countries with the highest number of tsunami events
2. Tsunami event clustering
3. Evolution of tsunami events over time
4. Geographic distribution of tsunami events

## Notes
- The dataset combines temporal, spatial and analytical dimensions.
- Historical information may be incomplete for some events.
- The results should be interpreted as descriptive analyses rather than causal relationships.
